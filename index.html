Documentación de Integración de Manus AI en Blackboard

Este documento proporciona la información necesaria y una guía para integrar Manus AI con Blackboard utilizando el estándar LTI 1.3.

1. Requisitos Previos

Para una integración exitosa, asegúrese de tener lo siguiente:

  • Acceso de administrador a su instancia de Blackboard Learn.
  • Una cuenta de desarrollador de Blackboard para registrar la aplicación LTI.
  • Una clave API de Manus AI (solicite acceso en el sitio web de Manus AI).
  • Conocimiento básico de la configuración LTI 1.3.

2. Datos de Configuración de Manus AI

Para la integración, necesitará la siguiente información de Manus AI:

  • Clave API de Manus AI: YOUR_MANUS_KEY (Obtenida a través del proceso de solicitud de acceso en el sitio web de Manus AI).
  • URL de Inicio de Sesión (Login URL): (Esta URL será proporcionada por Manus AI para la integración LTI. Si no está disponible, se debe consultar con el soporte de Manus AI).
  • URL(s) de Redirección de Herramientas (Tool Redirect URL(s)): (Estas URLs también serán proporcionadas por Manus AI y son necesarias para el flujo de autenticación LTI).
  • URL de la herramienta JWKS (JWKS Tool URL): (Esta URL es para el acceso a la clave pública de la herramienta y es crucial para la verificación de la firma LTI 1.3. Debe ser proporcionada por Manus AI).

Nota: La documentación de la API de Manus AI (https://manus-ai.com/api-docs) proporciona detalles sobre cómo interactuar con la API para funcionalidades avanzadas, pero para la integración LTI con Blackboard, las URLs específicas de LTI (Inicio de Sesión, Redirección, JWKS) son primordiales y deben ser obtenidas directamente de Manus AI.

3. Configuración de LTI 1.3 en Blackboard

Siga estos pasos para registrar Manus AI como una nueva aplicación LTI 1.3 en su instancia de Blackboard:

  1. Acceda al Panel de Administrador de Blackboard: Inicie sesión en su cuenta de administrador de Blackboard Learn.
  2. Navegue a Proveedores de Herramientas LTI: En el Panel de Administrador, bajo la sección 'Integración', seleccione 'Proveedores de Herramientas LTI'.
  3. Registre una nueva herramienta LTI 1.3/Advantage: Haga clic en 'Registrar Herramienta LTI 1.3/Advantage'.
  4. Complete el Formulario de Registro de la Aplicación: Utilice la siguiente información para rellenar los campos del formulario (referencia a la imagen proporcionada por el usuario):
    • Nombre de la aplicación: Manus IA
    • Descripción: Integración de Manus IA para mejorar la experiencia de aprendizaje con capacidades de IA avanzadas. (Puede personalizar esta descripción).
    • Dominio(s): Ingrese los dominios proporcionados por Manus AI. Estos son los dominios desde los cuales se originarán las solicitudes LTI. Separe múltiples dominios con comas.
    • Mi integración es compatible con LTI 1.3: Asegúrese de que este interruptor esté activado.
    • URL de inicio de sesión: Ingrese la URL de inicio de sesión proporcionada por Manus AI.
    • URL(s) de redirección de herramientas: Ingrese las URL(s) de redirección proporcionadas por Manus AI. Separe múltiples URLs con comas.
    • URL de la herramienta JWKS: Ingrese la URL de la herramienta JWKS proporcionada por Manus AI. Esta URL es crucial para la validación de la firma.
    • Algoritmo de firma: Seleccione RS256 (como se muestra en la imagen proporcionada).
    • Parámetros personalizados: Si Manus AI requiere parámetros personalizados adicionales, ingréselos aquí según sus especificaciones.
  5. Registre la solicitud: Una vez que todos los campos obligatorios estén completos, haga clic en 'Registrar solicitud'.

4. Especificaciones Técnicas

La integración de Manus AI con Blackboard se basa en el estándar LTI 1.3 (Learning Tools Interoperability 1.3), que es la versión más reciente y segura del protocolo LTI. Esto asegura una comunicación robusta y autenticada entre Blackboard y Manus AI.

4.1. Flujo de Autenticación

La autenticación se realiza mediante OAuth 2.0 y MTLS (Mutual Transport Layer Security), lo que garantiza que tanto Blackboard como Manus AI se autentiquen mutuamente antes de intercambiar información. El algoritmo de firma utilizado es RS256, un algoritmo de firma asimétrica que utiliza un par de claves pública/privada para asegurar la integridad y autenticidad de los mensajes LTI.

4.2. Intercambio de Datos

Manus AI, como agente de IA general, puede procesar y generar datos de diversos tipos (texto, imagen, audio, video) y coordinar con múltiples herramientas. La integración LTI permitirá el lanzamiento contextual desde Blackboard a Manus AI, facilitando el acceso a las capacidades de IA de Manus AI dentro del entorno de aprendizaje.

4.3. Consideraciones de Seguridad

  • Cifrado de Datos: Manus AI utiliza AES-256 y rotación de claves cuántica-segura para el cifrado de datos, lo que proporciona un alto nivel de seguridad para la información intercambiada.
  • Cumplimiento: Manus AI ofrece modos de cumplimiento GDPR/HIPAA, lo que es crucial para entornos educativos que manejan datos sensibles de estudiantes.
  • Manejo de Errores: La API de Manus AI incluye inteligencia para el manejo de errores, lo que puede ayudar a diagnosticar y mitigar problemas durante la integración y el uso.

5. Conclusión

La integración de Manus AI en Blackboard a través de LTI 1.3 ofrece una oportunidad para enriquecer la experiencia educativa con capacidades avanzadas de inteligencia artificial. Siguiendo los pasos y proporcionando la información correcta de Manus AI, los administradores pueden habilitar esta poderosa herramienta para sus usuarios de Blackboard.

Para cualquier problema o consulta adicional, se recomienda consultar la documentación oficial de Manus AI y el soporte de Blackboard.
